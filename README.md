# 📊 Election Data Analysis Dashboard (Power BI)

## Overview
Interactive Power BI dashboard analyzing election data (2014–2024) at booth level.

## Features
- Vote % trends
- Booth performance comparison
- Vote swing analysis
- Booth strength classification
- Votes required to win

## Tools
Power BI | DAX | Excel
