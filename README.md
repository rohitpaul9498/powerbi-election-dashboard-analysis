# 📊 Election Data Analysis Dashboard (Power BI)

## 🔍 Overview
Developed an interactive Power BI dashboard to analyze multi-year election data (2014–2024) at booth level.

This project focuses on identifying voting trends, performance variations, and data-driven insights using visualization techniques.

---

## 🚀 Key Features
- 📈 Vote % trend analysis across multiple years
- 🏢 Booth-level performance comparison
- 🔄 Vote swing analysis (voter movement tracking)
- 🧠 Booth strength classification (Strong / Competitive / Weak)
- 🎯 Votes required to win (gap analysis)
- 🎛 Interactive slicers for dynamic filtering

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Microsoft Excel
